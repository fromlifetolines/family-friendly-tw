# 親子友善台灣 Family-Friendly Taiwan

一個展示台灣親子友善場所的互動式網站，包含百貨公司、公園、博物館、交通樞紐等地點的詳細資訊。

## 🌟 特色功能

- 📍 台灣與日本親子友善場所資訊 (新增日本地區！🇯🇵)
*   🏢 包含百貨公司、Outlet、公共設施、主題樂園
*   🚆 主要交通樞紐 (台鐵、高鐵、機場捷運、Global Mall)
*   🔄 支援台灣/日本地區快速切換
*   🖼️ 高品質場所圖片
*   🔗 外部網站連結與購票優惠

## 🚀 網站連結

線上網站：[https://fromlifetolines.github.io/family-friendly-tw/](https://fromlifetolines.github.io/family-friendly-tw/)

## 💻 本地開發

### 安裝相依套件

```bash
npm install
```

### 啟動開發伺服器

```bash
npm run dev
```

開啟瀏覽器訪問 `http://localhost:5173/`

### 建置專案

```bash
npm run build
```

建置後的檔案會產生在 `dist` 資料夾中。

### 預覽建置結果

```bash
npm run preview
```

## 📦 技術棧

- ⚛️ React 19
- ⚡ Vite
- 📘 TypeScript
- 🎨 CSS

## 📝 自動部署

本專案使用 GitHub Actions 自動部署到 GitHub Pages。每次推送到 `main` 分支時，會自動建置並部署最新版本。
